# Self-Hosted

Run Kumbukum on infrastructure you control. The self-hosted edition is fully open source and includes the same product features as [Kumbukum Cloud](/cloud/).

## System Requirements

- **Node.js** >= 24
- **Docker** & Docker Compose (recommended)
- **pnpm** 10+
- **MongoDB** 6+
- **Redis** 6+
- **Typesense** 31+
