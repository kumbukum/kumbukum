# Browser Extension

Save URLs, create notes, or both — directly from any webpage without leaving your browser.

## Features

- **Save URL** — Bookmark the current page to Kumbukum with one click. The title, description, and content are extracted automatically.
- **Create Note** — Write a quick note from any webpage and save it to your Kumbukum account.
- **URL + Note** — Save the current page as a URL and attach a note to it at the same time.

## Supported Browsers

| Browser | Download |
| ------- | -------- |
| Chrome | [Chrome Web Store](https://chrome.google.com/webstore/detail/gnpfejkkamggccmpdpbddjhinkehnchf) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org) |

## Setup

1. Install the extension from your browser's store
2. Click the Kumbukum icon in the toolbar
3. Sign in with your Kumbukum account
4. Start saving URLs and notes from any page

## Multiple Accounts

You can add more than one Kumbukum account to the browser extension and switch between them without signing out.

### Adding an account

1. Click the Kumbukum icon in the toolbar
2. Open the account menu
3. Select **Add Account**
4. Sign in with a different Kumbukum account

### Switching accounts

Use the account menu to switch between your accounts. The active account determines where new URLs and notes are saved.

### Removing an account

Open the account menu, select the account you want to remove, and choose **Remove Account**.
